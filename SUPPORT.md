# Getting Help

If you need help with Kite, here are the best ways to get support:

## Bug Reports

If you've found a bug, please [open a bug report](https://github.com/kafkade/kite/issues/new?template=bug_report.yml) using our issue template. Include your OS, terminal emulator, Kite version, and steps to reproduce.

## Feature Requests

Have an idea for Kite? [Open a feature request](https://github.com/kafkade/kite/issues/new?template=feature_request.yml) and tell us about it.

## Questions and Discussions

For general questions, usage help, or project discussions, please use [GitHub Discussions](https://github.com/kafkade/kite/discussions) (once enabled).

## Security Issues

If you've found a security vulnerability, **do not open a public issue**. Please see [SECURITY.md](SECURITY.md) for responsible disclosure instructions.

## Contributing

Want to contribute code or documentation? See [CONTRIBUTING.md](CONTRIBUTING.md) for our contribution guide.
