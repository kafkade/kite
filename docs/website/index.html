Capabilities

Everything you need to
monitor your system.

Real-time metrics with an interactive, keyboard-driven interface. Built for engineers who live in the terminal.

⚡

CPU Monitoring

Total and per-core utilization with braille sparkline graphs, frequency tracking, and load averages — all updating in real time.

🧠

Memory & Swap

RAM and swap usage with historical graphs, bar gauges, and detailed breakdowns. Watch memory pressure as it happens.

💾

Disk I/O

Filesystem usage, read/write throughput rates, and mount point details. Spot I/O bottlenecks instantly.

🌐

Network Traffic

Per-interface bandwidth graphs with auto-scaling units. Monitor upload and download rates across all network interfaces.

📋

Process Management

Interactive process table with sorting, filtering, tree view, and signal control. Kill, stop, or renice processes directly.

⌨️

Keyboard-First

Vim-style navigation, modal input, settings menu, and help overlay. Every action is a keystroke away — no mouse needed.

Get Started

Up and running in seconds.

Clone the repo, build with Cargo, and launch. Requires Rust 1.85+.

Step 01

Clone & build

$ git clone https://github.com/kafkade/kite.git $ cd kite $ cargo build --release
Step 02

Launch

$ ./target/release/kite # Or with a custom refresh interval $ ./target/release/kite --interval 500
